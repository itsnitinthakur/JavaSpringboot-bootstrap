About blogit

This website is to collect data from the client by simply filing the form. To collect valid data we have used validation during filing up the form.To add data you have to click on Add user it takes you to client form. Once you submit your form you can see the success alert in the website. once you see that green alert now you can go to User List to see the client data in tables where you can the saving and personal information submitted by clients.

Technologies used

Frontend of this website is done by using bootstrap and html5.In the backend Springboot java Framework is in use.For save the client data we are using Mysql database which is running on a AWS on a server.If you want to see the code you can go through the code below
